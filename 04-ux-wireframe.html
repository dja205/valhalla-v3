Context

The payment gateway module uses a legacy Stripe integration that doesn't support modern webhook signatures or idempotency keys. This causes occasional duplicate charges and difficult reconciliation.

Acceptance Criteria

#gh-44 2026-03-26
Real-time Agent Pipeline Dashboard
portfolio/valhalla-v3
▼

Context

The Norse Agent System currently has no visibility layer. Operators cannot see which agent is running, what stage the pipeline is in, or how much cost has been incurred in the current session.

Acceptance Criteria

  • React dashboard with 5 views: Home, Backlog, Pipeline, Completed, Analytics
  • Real-time polling (10s interval) from execution-log.json
  • Dark space theme with amber/cyan accent palette
  • Mobile-responsive with bottom nav on mobile, sidebar on desktop
  • Stage cards with live duration counters and cost display
  • Job detail overlay with per-stage accordion and rendered markdown artefacts

Technical Constraints

  • Vite + React 18 + TypeScript + Tailwind CSS
  • No server-side rendering — static SPA
  • Must run on port 3000 via npm run dev
